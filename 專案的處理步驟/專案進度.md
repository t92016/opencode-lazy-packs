## 重大技術決策

1. **Chezmoi 為主要備份**：取代原本僅用 Google Drive 手動 zip 的方式，Chezmoi 管理 `~/.config/opencode/` 並透過 Git 同步到 GitHub
2. **PAT 以環境變數管理**：`opencode.jsonc` 中的 GitHub PAT 改為 chezmoi template `{{ env "..." }}` 讀取環境變數，避免 Git 儲存敏感資訊
3. **雙軌備份並行**：Chezmoi（主要）+ Google Drive（輔助，保留 3 天），新電腦優先走 Chezmoi 還原，Google Drive 為備援
4. **專案進度交接制**：每個專案根目錄的 `AGENTS.md` 只放專案介紹與設定，實際進度寫在 `專案的處理步驟/專案進度.md`

## 踩坑記錄

- **7/17 備份路徑分裂**：Google Drive 備份路徑曾出現 3 個不同版本，導致同日備份散落各地。已在 shutdown 技能中嚴格鎖定固定路徑 `G:\我的雲端硬碟\AI\opencode-config-backup\`
- **Chezmoi template 引號問題**：`opencode.jsonc.tmpl` 中的 `\"` 是 JSON 跳脫，但 Go template 不認得，需改為無跳脫的 `"` 才能正確解析
