    🔄 異機同步指南

    新電腦 6 步驟 + Agent 自動接續

    👤 你要做的

    1. 安裝 Node.js
    2. npm install -g opencode-ai
    3. gh auth login
    4. nlm login
    5. Google Drive 同步專案 + backup
    6. 還原 backup zip
    7. 補 OpenAI API Key
    8. cd 專案 → opencode → 說「開工」

    🤖 Agent 會做的

    1. 讀 AGENTS.md（專案全貌）
    2. 載入 startup 技能
    3. 讀 Obsidian（上次進度）
    4. 讀 Git（最新狀態）
    5. 給你完整進度摘要
    6. 6 個 MCP 全部隨時可用

    ☁️ 備份與還原

    設定檔自動備份到 Google Drive

    📤 備份（shutdown 自動）

    ~/AppData/Roaming/npm/node_modules/
  open-computer-use/          ← 持續整合
  J:...opencode-config-backup/
    └── opencode-config-latest.zip
        ├── opencode.jsonc    ← MCP 設定 + 權限
        └── skills/           ← 4 個全域技能

    📥 還原（新電腦手動）

    解壓 latest.zip → ~/.config/opencode/
nlm login          ← Google 認證
補 OpenAI Key      ← ~/.openai.env
opencode → 說「開工」
